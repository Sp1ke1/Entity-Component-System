Detailed Description

template<typename ObjectType>
class ObjectManager< ObjectType >

Manages objects of a specific ObjectType.

The ObjectManager class is responsible for creating, removing, and accessing objects of a specific ObjectType. It's a wrapper around PackedArray class with additional IObjectManager inheritance for polymorphism

Template Parameters
ObjectTypeThe type of objects managed by this manager.
