Detailed Description

Manages components in the Entity-Component-System (ECS) framework.

The ComponentManager class is responsible for registering, adding, removing, and accessing components in the ECS framework. It uses an unordered_map to generic ObjectManagers to store and manage component arrays efficiently. The class provides both regular and a safe interface with additional checks for managing components.
