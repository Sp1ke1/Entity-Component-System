Detailed Description

Entity Component System (ECS) manager.

The ECS class serves as the central manager for the Entity Component System framework. It provides an interface for creating, removing, and accessing entities and their components. The class also allows registering and running systems within the ECS framework. It provides both a regular interface and a safe interface with additional checks.
